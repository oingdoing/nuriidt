OCR API

각 기관에서 서비스 중인 고문헌 이미지 서비스에 당사에서 제공하는 OCR API 서비스 연동을 통해 각 기관의 기존 이미지 뷰어에서 바로 텍스트를 추출하여 원문 이미지와 OCR 결과 텍스트를 함께 제공, 원문 내용을 정확하게 파악하는데 활용할 수 있습니다.

적용 사례

KRpia에서는 11종의 콘텐츠에서 한자 원문 이미지를 텍스트로 자동 변환하여 열람할 수 있는 OCR 서비스를 제공하고 있습니다.

11종 콘텐츠 : 경국대전, 경세유포, 대전회통(대전회통_역주), 대한계년사, 사기, 연행록총간증보판, 한국과학기술사자료대계, 의문보감, 의방신감, 의종손익, 향약집성방
