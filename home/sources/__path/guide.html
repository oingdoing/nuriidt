  • Sprite Icon

    • 아이콘은 기본 이미지와 두배 이미지를 같이 sprite 폴더에 넣는다. 정확히 2배여야 한다. 예)16x16, 32x32
    • 두배 이미지는 같은 이름을 쓰되, 뒤에 @2x를 붙인다. 예) imges@2x.png

  • Button

    • button은 .app-button의 클래스를 공통으로 한다.
    • button의 텍스트는 span.button-content 안에 담는다.
    • 버튼 내부에 아이콘이 있을 경우, .button-content와 동일 레벨에 작성한다.
    • 단독의 영역을 가질 때는 .button-area로 감싼다.
    • 좌우로 벌어지는 두 개의 버튼 레이아웃은 .button-area에 .align-justify 클래스를 추가한다.

    신청 완료

    영업일 10일 이내에 연락처 또는 이메일로 개별 연락드리겠습니다.

    STEP 01.

    신청자 정보를
    알려주세요.

    • page의 제목은 그라데이션이 들어간 sub title을 h4로, main title을 h3으로 작성한다.
    • sub title의 텍스트는 span으로 감싼다.
    • 제목 하단 설명 텍스트가 들어갈 경우, h3(main title)과 동일 레벨로 p.app-pg__tit-desc에 작성한다.
    • 기본 좌측 정렬이며 중앙 정렬 시에는 .app-pg__tit 에 .align-center 클래스를 추가한다.
